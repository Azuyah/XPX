# v1.0.2

Full XPX release with the complete public feature set.

## Includes

- persistent XP and levels
- ranks, top list, tags, and level rewards
- missions and achievements
- stats and weapon tracking
- Credits economy
- shop, crates, tokens, and temporary XP boosts
- RTV, voting, and workshop map support
- admin tools for maps, modes, XP, credits, bots, and notifications
- special rounds and forced loadout modes

## Recommended For

- servers that want the full XPX experience
- communities using progression, missions, economy, and admin tooling together
