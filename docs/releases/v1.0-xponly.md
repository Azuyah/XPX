# v1.0-xponly

XP-only baseline release focused on core progression.

## Includes

- persistent XP
- level progression
- rank display
- tags and early core progression flow

## Does Not Include

- shop
- crates
- Credits economy
- missions
- achievements
- extended stats pages
- special rounds
- forced loadout modes
- newer admin economy and feature tooling

## Recommended For

- servers that only want the older XP/rank progression flow
- setups that do not need missions, economy, crates, or the newer feature systems
