# v1.0-xponly

Baseline XP-only release.

## Includes

- persistent XP
- level progression
- rank display
- tags and early core progression flow

## Does Not Include

- shop
- crates
- Credits economy
- missions
- achievements
- extended stats pages
- special rounds
- forced loadout modes
- newer admin economy and feature tooling

## Notes

- this is the renamed baseline formerly tagged as `v1.0-working`
- recommended if you want the older XP-focused milestone only
